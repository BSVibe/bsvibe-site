title: 시작하기
description: BSage에 가입하고 개인 AI 비서를 시작합니다.

BSage는 **온톨로지 기반 지식 관리 시스템**으로, 입력된 정보를 자동으로 분류하고 연결하는 AI 비서입니다. 다양한 소스에서 정보를 수집하고, 지식 간의 관계를 그래프로 시각화하며, 자연어 검색으로 필요한 지식을 즉시 찾을 수 있습니다.

## BSage가 하는 일

| 기능 | 설명 |
|------|------|
| **자동 지식 정리** | 입력된 정보를 자동으로 분류, 태깅, 연결 |
| **Knowledge Graph** | 지식 간의 관계를 시각적 그래프로 탐색 |
| **18개 플러그인** | Telegram, Slack, Discord, Email, RSS 등 다양한 외부 서비스 연동 |
| **자연어 검색** | 의미 기반(시맨틱) 검색으로 관련 지식을 정확하게 탐색 |
| **2nd Brain** | Seeds → Garden → Actions 3단계 지식 관리 |

## 빠른 시작

### 1단계: 계정 생성 및 로그인

1. 브라우저에서 [sage.bsvibe.dev](https://sage.bsvibe.dev)에 접속합니다.
2. **BSVibe 계정으로 로그인** 버튼을 클릭합니다.
   - 아직 BSVibe 계정이 없다면 [auth.bsvibe.dev](https://auth.bsvibe.dev)에서 회원가입합니다.
3. 로그인 후 대시보드가 표시됩니다.

![BSage 대시보드 — 지식 현황, 최근 활동, 플러그인 상태를 한눈에 확인할 수 있습니다.](/images/screenshots/bsage-dashboard.png)

대시보드에는 다음 정보가 표시됩니다:
- **Knowledge Stats**: 총 지식 항목 수, 카테고리별 분포
- **Recent Activity**: 최근 추가/수정된 지식 목록
- **Plugin Status**: 활성화된 플러그인 수와 연동 상태
- **Quick Actions**: 자주 사용하는 기능 바로가기

### 2단계: 플러그인 설정

BSage의 핵심은 **플러그인**을 통한 외부 서비스 연동입니다. Plugin Manager에서 필요한 플러그인을 활성화합니다.

1. 좌측 메뉴에서 **Plugin Manager**를 클릭합니다.
2. 사용할 플러그인을 찾아 **Enable** 버튼을 클릭합니다.
3. 플러그인에 필요한 credential을 입력합니다.
4. **Test Connection** 버튼으로 연결을 확인합니다.
5. **Save** 버튼을 클릭하여 설정을 저장합니다.

**추천 첫 설정 플러그인:**

| 플러그인 | 필요한 credential | 설정 난이도 |
|---------|-------------------|-------------|
| **Telegram** | Bot Token (BotFather에서 생성) | 쉬움 |
| **RSS** | 없음 (피드 URL만 입력) | 쉬움 |
| **Email** | IMAP 서버 정보, 이메일/비밀번호 | 보통 |
| **Slack** | Slack Bot Token (Slack App에서 생성) | 보통 |

:::tip[Telegram 플러그인 빠른 설정]
1. Telegram에서 [@BotFather](https://t.me/BotFather)와 대화합니다.
2. `/newbot` 명령으로 봇을 생성합니다.
3. 발급받은 Bot Token을 BSage Plugin Manager에 입력합니다.
4. 봇에게 메시지를 보내면 BSage에 자동으로 지식이 수집됩니다.
:::

### 3단계: 채팅으로 지식 입력

**Chat** 페이지에서 BSage와 대화를 시작합니다. 질문, 메모, 아이디어를 자유롭게 입력하면 BSage가 자동으로 처리합니다.

사용 예시:

| 입력 유형 | 예시 | BSage의 처리 |
|-----------|------|--------------|
| **메모** | "다음 주 월요일 디자인 리뷰 미팅" | 자동 태깅 (일정, 미팅), Seeds에 저장 |
| **아이디어** | "블로그에 AI 에이전트 비교 글 쓰기" | 자동 태깅 (아이디어, 콘텐츠), Actions 생성 |
| **질문** | "지난주에 저장한 API 설계 관련 메모 찾아줘" | Knowledge Graph에서 시맨틱 검색 후 결과 표시 |
| **링크** | "https://arxiv.org/abs/2401.12345 이 논문 정리해줘" | 링크 내용 요약, 태깅, Seeds에 저장 |
| **정보** | "PostgreSQL의 JSONB 인덱스는 GIN이 효율적" | 자동 분류 (기술, DB), 관련 지식과 연결 |

### 4단계: Knowledge Graph 탐색

**Knowledge Graph** 페이지에서 수집된 지식 간의 관계를 시각적으로 탐색합니다.

- **노드 클릭**: 해당 지식 항목의 상세 내용 표시
- **엣지**: 지식 간 관계 (참조, 관련, 반박 등)
- **줌/팬**: 마우스 휠과 드래그로 그래프 탐색
- **필터**: 카테고리, 태그, 날짜 기준으로 표시 범위 제한
- **검색**: 자연어로 검색하면 관련 노드가 하이라이트됨

### 5단계: 지식 활용

수집된 지식을 다양한 방식으로 활용합니다:

- **자연어 검색**: Chat에서 질문하면 관련 지식을 찾아 답변합니다.
- **요약 생성**: 특정 주제에 대한 지식을 모아 요약본을 생성합니다.
- **Actions 관리**: 지식에서 도출된 실행 항목을 추적합니다.
- **내보내기**: 지식을 Markdown, JSON 등으로 내보냅니다.
