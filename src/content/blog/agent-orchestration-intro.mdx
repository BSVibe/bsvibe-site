# AI 에이전트 오케스트레이션 입문

"AI 에이전트"라는 말이 어디서나 들립니다. 하지만 정확히 무엇이고, 왜 필요한 걸까요?

## AI 에이전트 = 역할이 있는 AI

ChatGPT에게 질문하면 답을 합니다. 하지만 그것은 **도구**입니다.

AI 에이전트는 다릅니다. 목표를 받으면 스스로 계획을 세우고, 필요한 도구를 사용하며, 결과를 판단합니다. 마치 신입 직원에게 업무를 맡기는 것과 비슷합니다.

## 왜 오케스트레이션인가

한 명의 에이전트로는 복잡한 일을 하기 어렵습니다. 마치 한 명이 회사의 모든 일을 할 수 없는 것처럼.

**오케스트레이션**은 여러 에이전트에게 역할을 나누고, 협력하게 만드는 것입니다.

- **기획 에이전트**: 프로젝트를 분석하고 작업을 나눔
- **개발 에이전트**: 코드를 작성하고 테스트
- **검토 에이전트**: 결과물을 검토하고 피드백
- **배포 에이전트**: 완성된 결과물을 전달

이것이 BSNexus가 하는 일입니다.

## BSNexus의 접근

BSNexus는 코딩 없이도 에이전트 팀을 구성할 수 있도록 설계했습니다.

### 조직도처럼 에이전트를 배치합니다

실제 회사의 조직도처럼 에이전트들의 관계를 설정합니다. 상위 에이전트가 하위 에이전트에게 업무를 분배하고, 결과를 수집합니다.

### 목표만 주면 알아서 실행합니다

"이번 주 뉴스레터를 만들어줘"라고 하면:
1. 리서치 에이전트가 최신 트렌드를 수집
2. 작성 에이전트가 초안을 작성
3. 편집 에이전트가 톤과 문법을 다듬음
4. 결과물이 당신의 메일로 전달

### 실패하면 다시 시도합니다

에이전트가 실패하면 자동으로 재시도하거나 대체 방법을 찾습니다. 사람이 개입할 필요가 최소화됩니다.

## 누구를 위한 도구인가

BSNexus는 **개발자가 아닌 사람**도 사용할 수 있도록 만들고 있습니다.

- PM이 반복적인 리포트를 자동화하고 싶을 때
- 디자이너가 참고 자료를 자동으로 수집하고 싶을 때
- 창업자가 시장 조사를 에이전트에게 맡기고 싶을 때

코드 한 줄 없이, 에이전트 팀을 만들고 실행하세요.
