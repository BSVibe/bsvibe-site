## Available Plugins

BSage provides 18 plugins.

### Input Plugins

| Plugin | Description |
|--------|-------------|
| Telegram | Receive messages via Telegram bot |
| Slack | Monitor Slack channels |
| Discord | Discord bot integration |
| Email | Receive emails via IMAP |
| RSS | Subscribe to RSS/Atom feeds |

### Process Plugins

| Plugin | Description |
|--------|-------------|
| Summarizer | LLM-powered automatic summarization |
| Tagger | Automatic tag assignment |
| DangerAnalyzer | Automatic risk detection |

### Output Plugins

| Plugin | Description |
|--------|-------------|
| S3 | Upload files to Amazon S3 |
| Vault | Obsidian Vault integration |

## Plugin Configuration

Enable each plugin and configure its credentials on the **Plugin Manager** page in the dashboard.
