## Overview

The BSNexus Kanban board updates in real time via WebSocket.

### Columns

| Column | Status | Description |
|--------|--------|-------------|
| Backlog | PENDING | Tasks waiting to be started |
| In Progress | IN_PROGRESS | Tasks currently being executed |
| Review | IN_REVIEW | Tasks awaiting review |
| Done | COMPLETED | Finished tasks |

### Task Card Details

Each card displays the following information:
- Task title and description
- Assigned Executor
- Dependencies
- Progress
