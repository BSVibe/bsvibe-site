BSupervisor는 AI 에이전트의 행동을 실시간으로 감시하고 감사하는 안전 레이어입니다.

## BSupervisor가 하는 일

- **실시간 규칙 평가**: 8개 기본 안전 규칙 (SQL injection, PII leak 등)
- **커스텀 규칙**: 조직에 맞는 감사 규칙 생성
- **비용 추적**: 에이전트별/모델별 LLM 비용 모니터링
- **일일 보고서**: 자동 생성되는 감사 리포트

## 빠른 시작

### 1. 계정 생성

[supervisor.bsvibe.dev](https://supervisor.bsvibe.dev)에서 BSVibe 계정으로 로그인합니다.

### 2. 에이전트 연결

**Settings** 페이지에서 감시할 AI 에이전트 플랫폼을 연동합니다:
- BSNexus (BSVibe 생태계 자동 연동)
- OpenAI / Anthropic (API 키 기반)
- 커스텀 에이전트 (Webhook)

### 3. 규칙 확인

**Rules** 페이지에서 8개 기본 안전 규칙이 자동 활성화되어 있습니다. 필요에 따라 규칙을 추가하거나 비활성화할 수 있습니다.

### 4. 대시보드 모니터링

**Dashboard**에서 이벤트 수, 규칙 위반, 비용 추이를 실시간으로 확인합니다.
