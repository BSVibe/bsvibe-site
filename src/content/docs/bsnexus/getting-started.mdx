BSNexus는 AI 에이전트들이 협업하여 소프트웨어를 만드는 오케스트레이션 플랫폼입니다.

## BSNexus가 하는 일

- **AI Architect**: 아이디어를 구조화된 프로젝트로 설계
- **자동 태스크 분해**: 프로젝트를 작업 단위로 자동 분할
- **병렬 실행**: 여러 AI 에이전트가 동시에 작업
- **실시간 Kanban**: 작업 진행 상황을 보드에서 확인

## 빠른 시작

### 1. 계정 생성

[nexus.bsvibe.dev](https://nexus.bsvibe.dev)에서 BSVibe 계정으로 로그인합니다.

### 2. 프로젝트 만들기

대시보드에서 **New Project** 버튼을 클릭하고 프로젝트 이름을 입력합니다.

### 3. Architect와 대화하기

**Architect** 페이지에서 만들고 싶은 것을 자연어로 설명합니다:

> "할 일 관리 앱을 만들어줘. React 프론트엔드와 FastAPI 백엔드로."

Architect가 요구사항을 분석하고 태스크를 생성합니다.

### 4. 진행 상황 확인

**Board** 페이지의 Kanban 보드에서 태스크 진행을 실시간으로 확인합니다.
