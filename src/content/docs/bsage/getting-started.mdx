BSage는 온톨로지 기반 지식 관리 시스템으로, 정보를 자동으로 분류하고 연결하는 AI 비서입니다.

## BSage가 하는 일

- **자동 지식 정리**: 입력된 정보를 자동 분류 및 태깅
- **Knowledge Graph**: 지식 간의 관계를 시각적으로 탐색
- **18개 플러그인**: Telegram, Slack, Discord, Email 등 다양한 연동
- **자연어 검색**: 의미 기반 검색으로 관련 지식 탐색

## 빠른 시작

### 1. 계정 생성

[sage.bsvibe.dev](https://sage.bsvibe.dev)에서 BSVibe 계정으로 로그인합니다.

### 2. 플러그인 설정

**Plugin Manager** 페이지에서 사용할 플러그인을 활성화합니다:
1. 원하는 플러그인의 **Enable** 클릭
2. 필요한 credential 입력 (예: Telegram Bot Token)
3. 설정 저장

### 3. 채팅 시작

**Chat** 페이지에서 BSage와 대화를 시작합니다. 질문, 메모, 아이디어를 자유롭게 입력하면 BSage가 자동으로 분류하고 저장합니다.

### 4. 지식 탐색

**Knowledge Graph** 페이지에서 수집된 지식 간의 관계를 시각적으로 탐색합니다.
