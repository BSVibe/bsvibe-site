title: 시작하기
description: BSGateway에 가입하고 첫 번째 API 호출을 합니다.

BSGateway는 LLM API 요청의 복잡도를 분석해 가장 비용 효율적인 모델로 자동 라우팅하는 프록시입니다.

## BSGateway가 하는 일

- **요청 복잡도 분석**: 간단한 질문은 저렴한 모델로, 복잡한 작업은 고성능 모델로
- **비용 최적화**: 동일한 품질을 유지하면서 LLM 비용을 절감
- **OpenAI 호환 API**: 기존 코드 변경 없이 엔드포인트만 교체

## 빠른 시작

### 1. 계정 생성

[gateway.bsvibe.dev](https://gateway.bsvibe.dev)에서 BSVibe 계정으로 로그인합니다.

### 2. 모델 등록

대시보드 **Models** 페이지에서 사용할 LLM 모델을 추가합니다:

1. **Add Model** 클릭
2. Provider 선택 (Anthropic, OpenAI, Ollama 등)
3. API 키 입력 — AES-256-GCM으로 암호화되어 저장됩니다
4. 모델 ID 설정 (예: `claude-sonnet`, `gpt-4o-mini`)

### 3. API 키 발급

대시보드 **API Keys** 페이지에서 BSGateway API 키를 생성합니다.

> API 키는 생성 시 한 번만 표시됩니다. 안전한 곳에 보관하세요.

### 4. API 호출

기존 OpenAI SDK 코드에서 base URL과 API 키만 변경하면 됩니다:

```python
from openai import OpenAI

client = OpenAI(
    base_url="https://gateway.bsvibe.dev/v1",
    api_key="your-bsgateway-api-key",
)

response = client.chat.completions.create(
    model="auto",  # BSGateway가 최적 모델 자동 선택
    messages=[{"role": "user", "content": "Hello!"}],
)
```

`model="auto"`로 요청하면 분류기가 복잡도를 판단해 최적 모델을 선택합니다.
특정 모델을 지정하려면 등록한 모델 ID를 사용하세요.
